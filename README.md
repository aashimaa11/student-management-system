# Student Management System (C++)

A scalable Student Management System built using C++ with Object-Oriented Programming and Standard Template Library. The system supports efficient data handling, grading, analytics, and persistent storage using file handling.

---

## Features

- Add, View, Update, and Delete student records (CRUD)
- Fast search using hash map (O(1) lookup)
- Sorting by name and percentage
- Marks, percentage, and grade calculation
- Undo delete functionality using stack
- Topper identification
- Analytics (average percentage, fail count)
- Persistent storage using file handling

---

## Tech Stack

- C++
- OOP (Classes, Objects)
- STL (vector, unordered_map, stack)
- File Handling
